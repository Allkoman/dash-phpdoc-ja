dash-php
============

Generates a [Dash](http://kapeli.com/dash) docset for [PHP](http://php.net/).

Support multi-language documents (en/ja/ru/ro/es/tr/fr/de/zh/pt_BR)

&nbsp;

usage
============

cd "checkout dir"

php generate-php.php

&nbsp;

requirement
============

- hh.exe, msys (Windows)
- chmlib (Mac, Linux)
