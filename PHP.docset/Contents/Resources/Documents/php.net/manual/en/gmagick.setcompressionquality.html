Gmagick::setCompressionQuality

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Gmagick::setCompressionQuality — Sets the object's default compression quality

説明

Gmagick Gmagick::setCompressionQuality ( int $quality = 75 )

Sets the object's default compression quality.

パラメータ

quality

返り値

The Gmagick object on success

エラー / 例外

エラー時に GmagickException をスローします。

例

例1 Gmagick::setCompressionQuality()

<?php
$gm = new Gmagick();
$gm->read("magick:rose");
$gm->setCompressionQuality(2);
?>
