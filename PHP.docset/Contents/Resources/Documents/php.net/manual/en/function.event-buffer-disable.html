event_buffer_disable

(PECL libevent >= 0.0.1)

event_buffer_disable — バッファつきイベントを無効にする

説明

bool event_buffer_disable ( resource $bevent , int $events )

指定したバッファつきイベントを無効にします。

パラメータ

bevent

有効なバッファつきイベントリソース。

events

EV_READ と EV_WRITE の任意の組み合わせ。

返り値

event_buffer_disable() は、成功した場合に TRUE、エラー時に FALSE を返します。
