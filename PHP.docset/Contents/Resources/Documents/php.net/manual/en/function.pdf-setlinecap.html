PDF_setlinecap

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setlinecap — linecap パラメータを設定する

説明

bool PDF_setlinecap ( resource $p , int $linecap )

linecap を設定し、 ストロークを考慮したうえでパスの終端の形状を制御します。
