mysqli_stmt::$field_count

mysqli_stmt_field_count

(PHP 5, PHP 7)

mysqli_stmt::$field_count -- mysqli_stmt_field_count — 指定したステートメントのフィールド数を返す

説明

オブジェクト指向型

int $mysqli_stmt->field_count;

手続き型

int mysqli_stmt_field_count ( mysqli_stmt $stmt )
警告
