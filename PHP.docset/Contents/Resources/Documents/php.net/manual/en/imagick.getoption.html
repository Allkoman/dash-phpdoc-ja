Imagick::getOption

(PECL imagick 2.0.0)

Imagick::getOption — 指定したキーに対応する値を返す

説明

string Imagick::getOption ( string $key )

オブジェクト内の指定したキーに対応する値を返します。

パラメータ

key

オプションの名前。

返り値

指定したキーに対応する値を返します。

エラー / 例外

エラー時に ImagickException をスローします。
