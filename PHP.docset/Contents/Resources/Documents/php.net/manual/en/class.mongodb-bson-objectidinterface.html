The MongoDB\BSON\ObjectIdInterface interface

(mongodb >=1.3.0)

はじめに

This interface is implemented by MongoDB\BSON\ObjectId but may also be used for type-hinting and userland classes.

クラス概要

MongoDB\BSON\ObjectIdInterface {
/* メソッド */
abstract public integer getTimestamp ( void )
abstract public string __toString ( void )
}

目次

  • MongoDB\BSON\ObjectIdInterface::getTimestamp — Returns the timestamp component of this ObjectIdInterface
  • MongoDB\BSON\ObjectIdInterface::__toString — Returns the hexidecimal representation of this ObjectIdInterface
