Imagick::mapImage

(PECL imagick 2.0.0)

Imagick::mapImage — 画像の色を、参照画像の中の最も近い色に置き換える

説明

bool Imagick::mapImage ( Imagick $map , bool $dither )

パラメータ

map

dither

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
