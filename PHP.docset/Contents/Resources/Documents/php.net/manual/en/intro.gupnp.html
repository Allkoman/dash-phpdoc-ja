はじめに

GUPnP は、UPnP デバイスとコントロールポイントを作成するための、 オブジェクト指向オープンソース・フレームワークです。 そして、GObject と libsoup を用いて C で書かれます。 このエクステンションは GUPnP を隠蔽して、同様の API を提示します。 ソースコードおよび更なる資料は全て、 » 公式サイト から入手可能です。
