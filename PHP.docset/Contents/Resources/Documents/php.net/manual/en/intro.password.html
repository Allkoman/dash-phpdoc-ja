はじめに

パスワードハッシュ API は crypt() を手軽に使えるようにしたラッパーで、 パスワードの作成や管理を安全な方法で行えます。

このモジュールが使えるのは PHP 5.5.0 以降ですが、PHP 5.3.7 以降で使える » ユーザーランドでの実装 もあります。
