PDF_shfill

(PECL pdflib >= 2.0.0)

PDF_shfill — シェーディングで領域を塗りつぶす

説明

bool PDF_shfill ( resource $pdfdoc , int $shading )

シェーディングオブジェクトに基づいたシェーディングで、 領域を塗りつぶします。

この関数は、PDF 1.4 以降で使用可能です。
