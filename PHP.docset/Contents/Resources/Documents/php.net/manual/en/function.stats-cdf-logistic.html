stats_cdf_logistic

(PECL stats >= 1.0.0)

stats_cdf_logistic — ロジスティック分布のパラメータのいずれかを、その他のパラメータの値から計算する

説明

float stats_cdf_logistic ( float $par1 , float $par2 , float $par3 , int $which )

ロジスティック分布の累積分布関数、その逆関数、またはパラメータのいずれかを戻します。 戻り値の種類とパラメータ (par1, par2, par3) は which によって決められます。

次の表は、which ごとの戻り値とパラメータを示したものです。 CDF, x, mu, s は、この順に、ロジスティック分布の累積分布関数、確率変数、位置母数、尺度母数を表します。

戻り値とパラメータ
which 戻り値 par1 par2 par3
1 CDF x mu s
2 x CDF mu s
3 mu x CDF s
4 s x CDF mu

パラメータ

par1

第一引数

par2

第二引数

par3

第三引数

which

計算される値を決めるフラグ

返り値

which によって決まる、CDF, x, mu, s のいずれかの値を戻します。
