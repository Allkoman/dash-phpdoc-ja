MongoDB\BSON\Binary::__toString

(mongodb >=1.2.0)

MongoDB\BSON\Binary::__toString — Returns the Binary's data

説明

final public string MongoDB\BSON\Binary::__toString ( void )

このメソッドは次のメソッドのエイリアスです。 MongoDB\BSON\Binary::getData().

パラメータ

この関数にはパラメータはありません。

返り値

Returns the Binary's data.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。

例

例1 MongoDB\BSON\Binary::__toString() example

<?php

var_dump((string) new MongoDB\BSON\Binary('foo', MongoDB\BSON\Binary::TYPE_GENERIC));

?>

上の例の出力は以下となります。

string(3) "foo"
