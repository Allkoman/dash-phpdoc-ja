MongoDB\BSON\Regex::serialize

(mongodb >=1.2.0)

MongoDB\BSON\Regex::serialize — Serialize a Regex

説明

final public string MongoDB\BSON\Regex::serialize ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns the serialized representation of the MongoDB\BSON\Regex.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。
