XsltProcessor::getSecurityPrefs

(PHP >= 5.4.0)

XsltProcessor::getSecurityPrefs — セキュリティ設定を取得する

説明

public int XsltProcessor::getSecurityPrefs ( void )

セキュリティ設定を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

XSL_SECPREF_READ_FILE、 XSL_SECPREF_WRITE_FILE、 XSL_SECPREF_CREATE_DIRECTORY、 XSL_SECPREF_READ_NETWORK、 XSL_SECPREF_WRITE_NETWORK を含むビットマスクを返します。
