SVMModel::getNrClass

(PECL svm >= 0.1.5)

SVMModel::getNrClass — モデルが訓練されたクラスの個数を返す

説明

public int SVMModel::getNrClass ( void )

モデルが訓練されたクラスの個数を返します。 １つのクラスおよび複数の回帰モデルにたいしては、2 を返します。

パラメータ

この関数にはパラメータはありません。

返り値

整数のクラスの個数を返します
