SolrInputDocument::getFieldCount

(PECL solr >= 0.9.2)

SolrInputDocument::getFieldCount — 文書内のフィールド数を返す

説明

public int SolrInputDocument::getFieldCount ( void )

文書内のフィールド数を返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に整数、失敗した場合に FALSE を返します.
