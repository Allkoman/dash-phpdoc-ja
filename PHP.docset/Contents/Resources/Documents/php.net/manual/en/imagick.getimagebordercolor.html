Imagick::getImageBorderColor

(PECL imagick 2.0.0)

Imagick::getImageBorderColor — 画像の前景色を返す

説明

ImagickPixel Imagick::getImageBorderColor ( void )

画像の前景色を返します。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
