Gmagick::deconstructimages

(PECL gmagick >= Unknown)

Gmagick::deconstructimages — 画像間の特定のピクセルの差を返す

説明

public Gmagick Gmagick::deconstructimages ( void )

各画像をシーケンス内の次の画像と比較し、 最大の境界範囲を返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に新しい Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
