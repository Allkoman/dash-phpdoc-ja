Ds\Set::copy

(PECL ds >= 1.0.0)

Ds\Set::copy — Returns a shallow copy of the set.

説明

public Ds\Set Ds\Set::copy ( void )

Returns a shallow copy of the set.

パラメータ

この関数にはパラメータはありません。

返り値

Returns a shallow copy of the set.

例

例1 Ds\Set::copy() example

<?php
$a = new \Ds\Set([1, 2, 3]);
$b = $a->copy();

// Updating the copy doesn't affect the original
$b->add(4);

print_r($a);
print_r($b);
?>

上の例の出力は、 たとえば以下のようになります。

Ds\Set Object
(
    [0] => 1
    [1] => 2
    [2] => 3
)
Ds\Set Object
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 4
)
