SplDoublyLinkedList::rewind

(PHP 5 >= 5.3.0, PHP 7)

SplDoublyLinkedList::rewind — イテレータを先頭に巻き戻す

説明

public void SplDoublyLinkedList::rewind ( void )

イテレータを先頭に巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

lincoln dot du dot j at gmail dot com 10-Jul-2017 05:42
$dll = new SplDoublyLinkedList();
$dll->push(2);
$dll->push(3);
$dll->unshift(5);
$dll->rewind(); // starting from beginning without this it will not work

while($dll->valid()){
    echo $dll->current()."\n";
    $dll->next();
}

//Output
5
2
3
