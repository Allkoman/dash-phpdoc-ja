ReflectionFunctionAbstract::isClosure

(PHP 5 >= 5.3.0, PHP 7)

ReflectionFunctionAbstract::isClosure — クロージャであるかどうかを調べる

説明

public bool ReflectionFunctionAbstract::isClosure ( void )

この関数が Closure であるかどうかを調べます。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

この関数が Closure である場合に TRUE、それ以外の場合に FALSE を返します。

例

例1 ReflectionFunctionAbstract::isClosure() の例

<?php
// クロージャではない
$function1 = 'str_replace';
$reflection1 = new ReflectionFunction($function1);
var_dump($reflection1->isClosure());

// クロージャである
$function2 = function () {};
$reflection2 = new ReflectionFunction($function2);
var_dump($reflection2->isClosure());
?>

上の例の出力は以下となります。

bool(false)
bool(true)
