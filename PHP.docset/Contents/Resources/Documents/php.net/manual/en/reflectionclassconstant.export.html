ReflectionClassConstant::export

(PHP 7 >= 7.1.0)

ReflectionClassConstant::export — Export

説明

public static string ReflectionClassConstant::export ( mixed $class , string $name [, bool $return ] )

Exports a reflection.

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

class

エクスポートするリフレクション。

name

The class constant name.

return

TRUE に設定すると、エクスポートを発行するのではなく返します。 FALSE (デフォルト) の場合はその逆です。

返り値
