getrandmax

(PHP 4, PHP 5, PHP 7)

getrandmax — 乱数の最大値を取得する

説明

int getrandmax ( void )

rand() をコールすることにより得られる最大の値を返します。

返り値

rand() が返す乱数の最大値を返します。

参考

  • rand() - 乱数を生成する
  • srand() - 乱数生成器を初期化する
  • mt_getrandmax() - 乱数値の最大値を表示する

The maximum value returned by getrandmax() is valid only if you call the rand() function without any parameter.

For eg

<?php
getrand(35000, 50000);
?>

would still work because you have specified the limits even though getrandmax() returned 32767.
