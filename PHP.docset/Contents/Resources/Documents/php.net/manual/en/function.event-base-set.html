event_base_set

(PECL libevent >= 0.0.1)

event_base_set — イベントベースをイベントと関連づける

説明

bool event_base_set ( resource $event , resource $event_base )

event_base を event と関連づけます。

パラメータ

event

有効なイベントリソース。

event_base

有効なイベントベースリソース。

返り値

event_base_set() は、成功した場合に TRUE、エラー時に FALSE を返します。
