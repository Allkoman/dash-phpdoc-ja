odbc_rollback

(PHP 4, PHP 5, PHP 7)

odbc_rollback — トランザクションをロールバックする

説明

bool odbc_rollback ( resource $connection_id )

この接続における全ての未解決の命令をロールバックします。

パラメータ

connection_id

ODBC 接続 ID。詳細は odbc_connect() を参照ください。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
