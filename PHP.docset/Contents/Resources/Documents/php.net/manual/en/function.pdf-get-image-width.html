PDF_get_image_width

(PHP 4, PECL pdflib >= 1.0.0)

PDF_get_image_width — 画像の幅を取得する [古い関数]

説明

この関数は PDFlib バージョン 3 で廃止されました。 かわりにパラメータ imagewidth を用いて PDF_get_value() を使用してください。
