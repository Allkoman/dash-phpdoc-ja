Throwable::getCode

(PHP 7)

Throwable::getCode — 例外コードを取得する

説明

abstract public int Throwable::getCode ( void )

スローされたオブジェクトのエラーコードを返します。

パラメータ

この関数にはパラメータはありません。

返り値

例外コードを返します。 Exception の場合は integer ですが、 その派生クラスでは別の型かもしれません (たとえば PDOException の場合は string です)。
