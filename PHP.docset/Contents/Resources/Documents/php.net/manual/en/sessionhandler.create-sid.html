SessionHandler::create_sid

(PHP 5 >= 5.5.1, PHP 7)

SessionHandler::create_sid — 新規セッション ID を返す

説明

public string SessionHandler::create_sid ( void )

新規セッション ID を生成して返します。

返り値

デフォルトのセッション・ハンドラに対して有効なセッションです。
