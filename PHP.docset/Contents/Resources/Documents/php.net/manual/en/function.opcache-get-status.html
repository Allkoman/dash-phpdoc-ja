opcache_get_status

(PHP 5 >= 5.5.5, PHP 7, PECL ZendOpcache > 7.0.2)

opcache_get_status — キャッシュについてのステータス情報を取得

説明

array opcache_get_status ([ boolean $get_scripts = TRUE ] )

この関数は、キャッシュ・インスタンスについてのステータス情報を返します。

パラメータ

get_scripts

スクリプト固有の状態の情報を含む。

返り値

情報の配列を返します。スクリプト固有の状態の情報をオプションで含みます。

エラー / 例外

opcache.restrict_api を使用中で、 かつ現在のパスがルール違反の場合、 E_WARNING が発生します。 ステータス情報は返されません。
