実行時設定

php.ini の設定により動作が変化します。

Readline 設定オプション
名前 デフォルト 変更可能 変更履歴
cli.pager "" PHP_INI_ALL PHP 5.4.0 以降で使用可能。
cli.prompt "\\b \\> " PHP_INI_ALL PHP 5.4.0 以降で使用可能。

以下に設定ディレクティブに関する 簡単な説明を示します。

cli.pager string

コマンドライン の出力を表示する外部ツール。

cli.prompt string

コマンドラインのプロンプト。
