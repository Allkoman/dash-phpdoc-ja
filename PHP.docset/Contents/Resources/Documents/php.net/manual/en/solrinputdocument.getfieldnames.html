SolrInputDocument::getFieldNames

(PECL solr >= 0.9.2)

SolrInputDocument::getFieldNames — 文書内のフィールドを全て含む配列を返す

説明

public array SolrInputDocument::getFieldNames ( void )

文書内のフィールドを全て含む配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功時には配列、失敗時には FALSE を返します。
