Ds\Deque::clear

(PECL ds >= 1.0.0)

Ds\Deque::clear — Removes all values from the deque.

説明

public void Ds\Deque::clear ( void )

Removes all values from the deque.

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

例

例1 Ds\Deque::clear() example

<?php
$deque = new \Ds\Deque([1, 2, 3]);
print_r($deque);

$deque->clear();
print_r($deque);
?>

上の例の出力は、 たとえば以下のようになります。

Ds\Deque Object
(
    [0] => 1
    [1] => 2
    [2] => 3
)
Ds\Deque Object
(
)
