定義済み定数

以下の定数が定義されています。 この関数の拡張モジュールが PHP 組み込みでコンパイルされているか、 実行時に動的にロードされている場合のみ使用可能です。

FORCE_GZIP (integer)
FORCE_DEFLATE (integer)
ZLIB_ENCODING_RAW (integer)
RFC 1951 の DEFLATE アルゴリズム。PHP 7.0.0 以降で利用可能です。
ZLIB_ENCODING_DEFLATE (integer)
RFC 1950 の ZLIB 圧縮アルゴリズム。PHP 7.0.0 以降で利用可能です。
ZLIB_ENCODING_GZIP (integer)
RFC 1952 の GZIP アルゴリズム。PHP 7.0.0 以降で利用可能です。
ZLIB_FILTERED (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_HUFFMAN_ONLY (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_FIXED (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_RLE (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_DEFAULT_STRATEGY (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_BLOCK (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_NO_FLUSH (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_PARTIAL_FLUSH (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_SYNC_FLUSH (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_FULL_FLUSH (integer)
PHP 7.0.0 以降で利用可能です。
ZLIB_FINISH (integer)
PHP 7.0.0 以降で利用可能です。
