MongoDB\BSON\ObjectId::jsonSerialize

(mongodb >=1.2.0)

MongoDB\BSON\ObjectId::jsonSerialize — Returns a representation that can be converted to JSON

説明

final public mixed MongoDB\BSON\ObjectId::jsonSerialize ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns data which can be serialized by json_encode() to produce the » extended JSON representation of the MongoDB\BSON\ObjectId.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。

  • JsonSerializable::jsonSerialize() - JSON にシリアライズしたいデータを指定する
  • json_encode() - 値を JSON 形式にして返す
