MongoClient::dropDB

(PECL mongo >=1.3.0)

MongoClient::dropDB — Drops a database [deprecated]

このメソッドを定義している拡張モジュールは非推奨です。 かわりに MongoDB 拡張モジュールを使うべきです。 このメソッドの代替は、新しい拡張モジュールには存在しません。しかし、かわりに使える PHP ライブラリ があります。

  • » MongoDB\Client::dropDatabase()

説明

public array MongoClient::dropDB ( mixed $db )
警告

Deprecated

Use MongoDB::drop() instead.

パラメータ

db

The database to drop. Can be a MongoDB object or the name of the database.

返り値

Returns the database response.
