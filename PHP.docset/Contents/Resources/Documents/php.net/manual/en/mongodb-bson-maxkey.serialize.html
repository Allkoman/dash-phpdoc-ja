MongoDB\BSON\MaxKey::serialize

(mongodb >=1.2.0)

MongoDB\BSON\MaxKey::serialize — Serialize a MaxKey

説明

final public string MongoDB\BSON\MaxKey::serialize ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns the serialized representation of the MongoDB\BSON\MaxKey.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。
