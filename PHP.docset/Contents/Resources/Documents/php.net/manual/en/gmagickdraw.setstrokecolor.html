GmagickDraw::setstrokecolor

(PECL gmagick >= Unknown)

GmagickDraw::setstrokecolor — オブジェクトの輪郭の描画に使う色を設定する

説明

public GmagickDraw GmagickDraw::setstrokecolor ( mixed $color )

オブジェクトの輪郭の描画に使う色を設定します。

パラメータ

color

輪郭の色を表す GmagickPixel オブジェクトあるいは文字列。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
