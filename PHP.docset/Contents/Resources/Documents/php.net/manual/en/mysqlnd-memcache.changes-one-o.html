PECL/mysqlnd_memcache 1.0 系

1.0.0-alpha

  • リリース日: 未定
  • 目標: SQL の SELECT から MySQL Memcache プラグインへの基本的なマッピング。

最初のリリースでは、基本的な SELECT 文を MySQL Memcache プラグインへのアクセスにマップします。 これによって、パフォーマンス的なメリットが得られるでしょう。 というのも、Memcache プロトコルを使って MySQL ストレージに直接 キー/バリュー 形式のアクセスをするので、SQL を使うよりも高速になります。
