Gmagick::getimagecompose

(PECL gmagick >= Unknown)

Gmagick::getimagecompose — 画像の合成演算子を返す

説明

public int Gmagick::getimagecompose ( void )

画像に関連付けられている合成演算子を返します。

パラメータ

この関数にはパラメータはありません。

返り値

画像に関連付けられている合成演算子を返します。

エラー / 例外

エラー時に GmagickException をスローします。
