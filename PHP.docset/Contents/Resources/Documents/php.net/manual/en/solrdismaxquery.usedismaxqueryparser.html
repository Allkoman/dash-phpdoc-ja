SolrDisMaxQuery::useDisMaxQueryParser — Switch QueryParser to be DisMax Query Parser

説明

public SolrDisMaxQuery SolrDisMaxQuery::useDisMaxQueryParser ( void )

Switch QueryParser to be DisMax Query Parser

パラメータ

この関数にはパラメータはありません。

返り値

SolrDisMaxQuery

例

例1 SolrDisMaxQuery::useDisMaxQueryParser() example

<?php

$dismaxQuery = new SolrDisMaxQuery();
$dismaxQuery->useDisMaxQueryParser();
echo $dismaxQuery;
?>

上の例の出力は、 たとえば以下のようになります。

defType=dismax
