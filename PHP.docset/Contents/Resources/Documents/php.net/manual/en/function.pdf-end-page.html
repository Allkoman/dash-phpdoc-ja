PDF_end_page

(PHP 4, PECL pdflib >= 1.0.0)

PDF_end_page — ページを終了する

説明

bool PDF_end_page ( resource $p )

ページを終了します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
