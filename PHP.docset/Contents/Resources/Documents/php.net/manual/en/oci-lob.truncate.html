OCI-Lob::truncate

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Lob::truncate — ラージオブジェクトを切りつめる

説明

bool OCI-Lob::truncate ([ int $length = 0 ] )

LOB を切りつめます。

パラメータ

length

指定した場合は、このメソッドはラージオブジェクトを length バイトに切りつめます。 そうでない場合は、LOB を完全に消去します。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
