SVMModel::save

(PECL svm >= 0.1.0)

SVMModel::save — モデルをファイルに保管

説明

public bool SVMModel::save ( string $filename )

後で使用するために、モデルデータをファイルに保管します。

パラメータ

filename

モデルを保管するファイル

返り値

エラー時に SVMException をスローします。 成功時に true を返します。
