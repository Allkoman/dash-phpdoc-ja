QuickHashIntHash::get

(PECL quickhash >= Unknown)

QuickHashIntHash::get — キーを指定してハッシュから値を取得する

説明

public int QuickHashIntHash::get ( int $key )

キーを指定してハッシュから値を取得します。

パラメータ

key

エントリのキー。

返り値

キーが存在する場合はその値、キーが存在しない場合は NULL を返します。

例

例1 QuickHashIntHash::get() の例

<?php
$hash = new QuickHashIntHash( 8 );
var_dump( $hash->get( 1 ) );

var_dump( $hash->add( 2 ) );
var_dump( $hash->get( 2 ) );

var_dump( $hash->add( 3, 5 ) );
var_dump( $hash->get( 3 ) );
?>

上の例の出力は、 たとえば以下のようになります。

bool(false)
bool(true)
int(1)
bool(true)
int(5)
