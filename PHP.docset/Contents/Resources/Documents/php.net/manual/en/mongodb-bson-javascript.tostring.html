MongoDB\BSON\Javascript::__toString

(mongodb >=1.2.0)

MongoDB\BSON\Javascript::__toString — Returns the Javascript's code

説明

final public string MongoDB\BSON\Javascript::__toString ( void )

このメソッドは次のメソッドのエイリアスです。 MongoDB\BSON\Javascript::getCode().

パラメータ

この関数にはパラメータはありません。

返り値

Returns the Javascript's code.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。

例

例1 MongoDB\BSON\Javascript::__toString() example

<?php

var_dump((string) new MongoDB\BSON\Javascript('function foo(bar) { return bar; }'));

?>

上の例の出力は以下となります。

string(33) "function foo(bar) { return bar; }"
