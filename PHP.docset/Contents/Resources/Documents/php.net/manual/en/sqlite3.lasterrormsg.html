SQLite3::lastErrorMsg

(PHP 5 >= 5.3.0, PHP 7)

SQLite3::lastErrorMsg — 直近で失敗した SQLite リクエストについての英文テキストの説明を返す

説明

public string SQLite3::lastErrorMsg ( void )

直近で失敗した SQLite リクエストについての英文テキストの説明を返します。

パラメータ

この関数にはパラメータはありません。

返り値

直近で失敗した SQLite リクエストについての英文テキストの説明を返します。
