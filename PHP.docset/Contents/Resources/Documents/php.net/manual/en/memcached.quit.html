Memcached::quit

(PECL memcached >= 2.0.0)

Memcached::quit — 開いている接続を閉じる

説明

public bool Memcached::quit ( void )

Memcached::quit() は、memcache サーバーとの接続をすべて閉じます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
