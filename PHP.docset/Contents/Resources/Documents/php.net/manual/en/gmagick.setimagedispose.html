Gmagick::setimagedispose

(PECL gmagick >= Unknown)

Gmagick::setimagedispose — 画像の配置方法を設定する

説明

public Gmagick Gmagick::setimagedispose ( int $disposeType )

画像の配置方法を設定します。

パラメータ

disposeType

画像の配置方法。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
