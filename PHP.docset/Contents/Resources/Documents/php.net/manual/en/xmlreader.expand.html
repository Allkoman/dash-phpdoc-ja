XMLReader::expand

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::expand — 現在のノードのコピーを DOM オブジェクトとして返す

説明

public DOMNode XMLReader::expand ([ DOMNode $basenode ] )

このメソッドは、現在のノードをコピーして適切な DOM オブジェクトを返します。

パラメータ

basenode

DOM オブジェクトを作る対象の DOMDocument を示す DOMNode。

返り値

DOMNode か、エラー時には FALSE を返します。

変更履歴

バージョン 説明
5.3.0 basenode パラメータが追加されました。
