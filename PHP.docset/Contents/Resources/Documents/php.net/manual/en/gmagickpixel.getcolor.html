GmagickPixel::getcolor

(PECL gmagick >= Unknown)

GmagickPixel::getcolor — 色を返す

説明

public mixed GmagickPixel::getcolor ([ bool $as_array = false [, bool $normalize_array = false ]] )

GmagickPixel オブジェクトの色を文字列あるいは配列で返します。 透明度が設定されているときは、リストの四番目の値として返します。

パラメータ

as_array

TRUE にすると、返り値を文字列ではなく配列にします。

normalize_array

TRUE にすると、色の値を正規化します。

返り値

各チャネルの値を文字列あるいは配列で返します。normalize_array に TRUE を市得治した場合は正規化をします。エラー時には GmagickPixelException をスローします。
