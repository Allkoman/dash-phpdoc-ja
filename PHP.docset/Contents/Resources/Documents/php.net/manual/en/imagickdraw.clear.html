ImagickDraw::clear

(PECL imagick 2.0.0)

ImagickDraw::clear — ImagickDraw をクリアする

説明

bool ImagickDraw::clear ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

ImagickDraw オブジェクト上のコマンドをすべてクリアし、 デフォルト設定で初期化します。

返り値

ImagickDraw オブジェクトを返します。
