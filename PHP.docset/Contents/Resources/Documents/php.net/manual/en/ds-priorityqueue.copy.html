Ds\PriorityQueue::copy

(PECL ds >= 1.0.0)

Ds\PriorityQueue::copy — Returns a shallow copy of the queue.

説明

public Ds\PriorityQueue Ds\PriorityQueue::copy ( void )

Returns a shallow copy of the queue.

パラメータ

この関数にはパラメータはありません。

返り値

Returns a shallow copy of the queue.

例

例1 Ds\PriorityQueue::copy() example

<?php
$queue = new \Ds\PriorityQueue();

$queue->push("a",  5);
$queue->push("b", 15);
$queue->push("c", 10);

print_r($queue->copy());
?>

上の例の出力は、 たとえば以下のようになります。

Ds\PriorityQueue Object
(
    [0] => b
    [1] => c
    [2] => a
)
