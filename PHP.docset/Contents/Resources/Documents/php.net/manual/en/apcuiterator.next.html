APCUIterator::next

(PECL apcu >= 5.0.0)

APCUIterator::next — Move pointer to next item

説明

public void APCUIterator::next ( void )

Moves the iterator pointer to the next element.

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
