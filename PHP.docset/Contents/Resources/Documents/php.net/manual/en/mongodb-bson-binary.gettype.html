MongoDB\BSON\Binary::getType

(mongodb >=1.0.0)

MongoDB\BSON\Binary::getType — Returns the Binary's type

説明

final public integer MongoDB\BSON\Binary::getType ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns the Binary's type.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。

例

例1 MongoDB\BSON\Binary::getType() example

<?php

$binary = new MongoDB\BSON\Binary('foo', MongoDB\BSON\Binary::TYPE_GENERIC);
var_dump($binary->getType());

?>

上の例の出力は以下となります。

int(0)
