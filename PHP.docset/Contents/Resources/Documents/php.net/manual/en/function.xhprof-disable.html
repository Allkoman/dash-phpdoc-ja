xhprof_disable

(PECL xhprof >= 0.9.0)

xhprof_disable — xhprof プロファイラを終了する

説明

array xhprof_disable ( void )

プロファイラを終了して、xhprof の実行結果データを返します。

パラメータ

この関数にはパラメータはありません。

返り値

実行結果の xhprof データを配列で返します。

例

例1 xhprof_disable() の例

<?php
xhprof_enable();

$foo = strlen("foo bar");

$xhprof_data = xhprof_disable();

print_r($xhprof_data);
?>

上の例の出力は、 たとえば以下のようになります。

Array
(
    [main()==>strlen] => Array
        (
            [ct] => 1
            [wt] => 279
        )

    [main()==>xhprof_disable] => Array
        (
            [ct] => 1
            [wt] => 9
        )

    [main()] => Array
        (
            [ct] => 1
            [wt] => 610
        )

)
