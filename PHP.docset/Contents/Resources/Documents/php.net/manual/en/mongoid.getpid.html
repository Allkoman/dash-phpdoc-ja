MongoId::getPID

(PECL mongo >= 1.0.11)

MongoId::getPID — プロセス ID を取得する

このメソッドを定義している拡張モジュールは非推奨です。 かわりに MongoDB 拡張モジュールを使うべきです。 このメソッドの代替は、新しい拡張モジュールには存在しません。

説明

public int MongoId::getPID ( void )

Mongo ID から pid を取り出します。

パラメータ

この関数にはパラメータはありません。

返り値

MongoId の PID を返します。
