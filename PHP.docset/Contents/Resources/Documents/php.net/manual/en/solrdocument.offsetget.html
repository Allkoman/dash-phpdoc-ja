SolrDocument::offsetGet

(PECL solr >= 0.9.2)

SolrDocument::offsetGet — フィールドを取得

説明

public SolrDocumentField SolrDocument::offsetGet ( string $fieldName )

これは、オブジェクトが配列として扱われる際に、フィールドを取得するために使用されます。

パラメータ

fieldName

フィールド名

返り値

SolrDocumentField オブジェクトを返します。
