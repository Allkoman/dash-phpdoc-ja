HaruPage::arc

(PECL haru >= 0.0.1)

HaruPage::arc — 現在のパスに弧を追加する

説明

bool HaruPage::arc ( float $x , float $y , float $ray , float $ang1 , float $ang2 )

現在のパスに弧を追加します。

パラメータ

x

中心の水平座標。

y

中心の垂直座標。

ray

弧の半径。

ang1

開始角。

ang2

終了角。ang1 より大きくなければなりません。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
